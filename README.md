# Calculadora Web - Leo Tech

**Tecnologia, Precisão e Inovação**

Uma calculadora web premium, moderna e responsiva desenvolvida com **HTML5, CSS3 e JavaScript puro**, oferecendo uma experiência visual futurista com desempenho profissional.

## 🚀 Características Principais

### Dois Modos de Operação

#### 1️⃣ **Modo Simples**
- Operações aritméticas básicas (+, −, ×, ÷)
- Percentual (%)
- Inversão de sinal (+/−)
- Limpeza (C) e exclusão (⌫)
- Ponto decimal (.)

#### 2️⃣ **Modo Científico**
- Todas as funções básicas
- **Potência**: x², x³, xʸ
- **Raízes**: √ (quadrada), ∛ (cúbica)
- **Logaritmos**: log (base 10), ln (natural)
- **Exponencial**: eˣ
- **Fatorial**: n!
- **Trigonometria**: sin, cos, tan, sin⁻¹, cos⁻¹, tan⁻¹
- **Constantes**: π, e
- **Módulo**: |x|
- **Memória**: MC, MR, M+, M−
- **Parênteses**: ( )
- **Conversão de Ângulos**: Graus ↔ Radianos
- **Notação Científica**: Automática para números muito grandes/pequenos

## 🚀 Como Usar

1. **Clone ou baixe o repositório**
```bash
git clone https://github.com/leotech-projects/calculadora-web-LeoProjects.git
```

2. **Abra no navegador**
- Simplesmente abra o arquivo `index.html` no seu navegador
- Ou acesse através de um servidor web local

3. **Use a calculadora**
- Modo Simples: Operações básicas
- Clique em **⚙ Científico** para modo avançado
- Use o teclado ou clique nos botões

## 🎯 Exemplos de Uso

### Cálculo Simples
- Digite: `5 + 3`
- Pressione: `=`
- Resultado: `8`

### Função Científica
- Digite: `9`
- Clique: `√`
- Resultado: `3`

### Trigonometria
- Digite: `90`
- Clique: `sin`
- Resultado: `1` (em graus)

### Histórico
- Clique em qualquer cálculo do histórico para restaurar o resultado

## 🎨 Customização

### Alterar Cores
Abra `style.css` e modifique as variáveis CSS:

```css
:root {
    --neon-blue: #00D4FF;    /* Azul neon */
    --neon-cyan: #00FFFF;    /* Cyan neon */
    --neon-purple: #8A2BE2;  /* Roxo */
}
```

### Adicionar Funções
Edite `script.js` na seção de funções científicas:

```javascript
function myNewFunction() {
    const current = parseFloat(state.currentInput);
    const result = /* seu cálculo */;
    state.currentInput = formatResult(result);
    updateDisplay();
}
```
